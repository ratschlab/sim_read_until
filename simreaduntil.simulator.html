simreaduntil.simulator

package documentation
(source)

Provides the simulator

Module channel Model a single channel as a finite-state machine of channel elements
Module channel_element Channel element within a channel, e.g., a read, a gap between reads, an unblock delay
Module channel_stats Tracking channel statistics, for a single or multiple channels
Package example_grpc Example of a gRPC server (to check that gRPC networking works)
Module pore_model Pore model to convert DNA sequence to ONT raw signal
Module readpool Read pool that returns reads when requested, e.g., from a generator or a file
Module readswriter Write finished reads, e.g., to a file or list
Module simfasta_to_seqsum Convert a FASTA file generated by the ONTSimulator into a sequencing_summary.txt file, based on the FASTA headers.
Module simulator Simulate a collection of channels to synchronously start, stop, run mux scans, issue ReadUntil API calls on the channels
Module simulator_client gRPC Client to connect to a gRPC server exposing an ONTSimulator
Module simulator_params Manage simulation parameters such as bps_per_second, chunk_size, gap_samplers
Module simulator_server gRPC server exposing an underlying ReadUntilDevice
Module utils Utilities for the simulator
