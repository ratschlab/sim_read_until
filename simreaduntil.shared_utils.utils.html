def force_eval_generator(gen): (source) ¶

Evaluate a generator to its first yield.

Generator functions (using yield in the function) are only run when the first element is requested. This may come with some overhead for the first element, e.g. to open a FASTA file. This function avoids it by evaluating the generator to its first yield.

Parameters
gena generator
Returns
new generator object (do not use the old one which has one element missing)
