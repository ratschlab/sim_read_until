class SingleFileReadsWriter(ReadsWriter): (source)

Constructor: SingleFileReadsWriter(reads_out_fh, prefix)

View In Hierarchy

Write reads to one file (by default stdout), appending reads to the file as they are written (possibly with buffering)

When pickling the file, the file is flushed. When reloading it, the filehandler is not restored and must be set directly. This class is useful for debugging by writing to sys.stdout.
