simreaduntil.simulator.channel_element

module documentation
(source)

Channel element within a channel, e.g., a read, a gap between reads, an unblock delay

Class ChannelBroken Infinite gap that indicates that the channel is completely broken and not just temporarily inactive
Class ChannelElement Base class for element/interval in channel [t_start, t_duration]
Class ChunkedRead Read divided into chunks that can progressively be read from
Class LongGap Long gap between reads due to temporary pore blockage (inactive period)
Class MuxScan Mux scan in channel, so the channel is effectively unusable
Class NoReadLeftGap Infinite gap because no more reads are left
Class ReadDescriptionParser Parse the read description field written by the ChunkedRead
Class ReadEndReason Reason why a read ended, stop receiving is not part of it
Class ReadTags Tags to attach to a read, multiple are possible!
Class ShortGap Short gap between reads in channel
Class UnblockDelay Gap due to unblocking of read
Function element_to_short_str short string representation of element
Function estimate_ref_len Estimate reference length of new sequence (cut short due to selective sequencing), not exact due to indels
Variable end_reason_to_ont_map map simulator read end reasons to real ONT device read end reasons
