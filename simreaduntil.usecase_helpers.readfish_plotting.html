def plot_extra_basecalling_delay_per_iter(df, save_dir=None): (source) ¶

Plots the extra basecalling delay per iteration.

One iteration is a called to get_basecalled_read_chunks() which logs the total time spent due to basecalling. The basecalling starts right when the function is called, so if the processing of the basecalled data takes longer, there is no extra delay due to basecalling.
