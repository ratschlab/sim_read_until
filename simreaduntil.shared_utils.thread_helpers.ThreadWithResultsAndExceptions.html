class ThreadWithResultsAndExceptions(threading.Thread): (source)

View In Hierarchy

Thread that executes a function, gives access to its return value, and saves and logs any exceptions.

This allows to check later whether a thread raised an exception, which otherwise may go unnoticed. Calling .join() raises an exception if an exception has occurred in the thread.
