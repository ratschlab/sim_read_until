Track coverage over time by parsing the NanoSim read ids

Class BasepairCoverageTracker Track coverage per chromosome in a genome or metagenome, given the read position (e.g. parsed from NanoSim id or from a PAF file)
Class BlockwiseCoverageTracker Track coverage per chromosome in a genome or metagenome in blocks, given the read position (e.g. parsed from NanoSim id or from a PAF file)
Class NanoSimCoverageTracker Track coverage by parsing the location from the NanoSim read ids
Class PafCoverageTracker Track coverage by looking up the read ids in a PAF file

def _add_count_to_blocked_array(arr, start, end, blocksize): (source) ¶

Increment count of region [start, end) in a block array

A block array keeps track of counts per block of size blocksize, the last block possibly being shorter. Each block is represented by a single integer and counts the number of basepairs covered by a read in that block.
